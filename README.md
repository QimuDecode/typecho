# WeTypecho--微信小程序
基于[WeTypecho](https://github.com/MingliangLu/WeTypecho)二开项目
> 暂不开源
## 支持的基本功能
- 首页预览置顶文章
- 评论，回复，点赞
- 图片视频显示
- 支持markdown,html解析
